# Weather App

Aplikacja pogodowa z funkcją kalkulacji energii fotowoltaicznej.

## Funkcjonalności
- Prognoza pogody na 7 dni
- Kalkulacja produkcji energii fotowoltaicznej
- Interaktywna mapa do wyboru lokalizacji
- Automatyczna geolokalizacja
- Tryb ciemny/jasny
- Responsywny design

## Technologie
- React
- TypeScript
- Tailwind CSS
- Spring Boot
- Open-Meteo API
- Leaflet Maps
